# Kerala-Weather-Prediction-Model
introduction:Weather prediction is essential for planning in agriculture, transport, and daily life.
This project predicts Kerala district weather using machine learning.

objective:Build a machine learning model to predict weather conditions.
Take district and date as input.
Provide future weather predictions in an easy interface.
Demonstrate real-world AI application in weather forecasting.

tech stack:
Python-Programming
Pandas,NumPy-Data Processing
scikit-learn-Machine Learning
Streamlit-Interactive Web Interface

Conclusion:it demonstrates how machine learning and data analysis can make weather forecasting smater and more localized
